Social Tools

Blogging isn't necessarily part of the job, but it doesn't hurt. I host my blog over at Wordpress.com. I like the service... I've tried most of the others and keep coming back to WP... Lately I've been using Windows Live Writer to post updates. I've tried using Word and the WP clients. I like WLW right now, so I use it.

Managing feeds and links I find interesting has been somewhat of a struggle for me. Right now I'm using Readly as my feed reader on Windows 8. On Windows Phone 8 I've been using Nextgen Reader. I use Pocket to save any links I'd like to spend more time with. The problem is that there isn't a Windows 8 app or Phone 8 app. I setup an email trigger at IFTT so that anything I email to my IFTT address will be added to Pocket. I was a fan of Instapaper, but for now I'm using Pocket. I use Delicious to archive the links that I favorite in Pocket...
